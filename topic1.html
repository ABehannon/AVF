Industry Research, Project Intent


  • What types of native features (on Android and iOS devices) would you be able to implement with the use of HTML/CSS/JS only?

  • What types of features do you think will pair well together in an app?

  • I think this really just depends on what you are trying to do. For example, I think geolocation and the camera go pretty well together. There are already other apps out there that show as such, giving you the ability to take a picture and mark where you took it.



  • Please describe any design and functionality intentions you may have for your Demo app components this month.

  • I would like to be able to use the map, geolocation to maybe mark areas of note down within the app for the user to keep information/tabs on. Working more into that, if possible what I'd really like to do is tie pictures to that, as well. Being able to use the camera, mark where it was taken or maybe being able to go to the picture, then use that picture as a store for GPS markers. We'll see what we can do, though.
